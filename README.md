# Gravitational Wave Detection Simulation 

## Description
The Gravitational Wave Detection Simulation is a web-based physics visualization project developed using HTML5 Canvas and JavaScript. It demonstrates how gravitational waves affect an interferometer by animating arm oscillations and displaying real-time wave signal graphs with noise for realism.

This project is designed to convert physics theory into interactive simulations for better understanding and educational use.

## Core Concepts Implemented

### HTML5 Canvas
Used for rendering graphics and real-time animations.

### Event Handling
JavaScript event listeners enable user interaction through sliders and buttons.

### Canvas Coordinate System
Uses top-left origin with downward positive Y-axis (different from Cartesian system).

## Simulation Features
- Interferometer visualization with two oscillating arms  
- Real-time gravitational wave animation  
- Live signal graph with noise simulation  
- Adjustable parameters:
  - Frequency
  - Amplitude
  - Noise Level
  - Phase Shift
  - Arm Length
  - Data Points
- Zoom and pause graph controls  
- Theoretical background section  

## Scientific Principle Used

The simulation is based on **Interferometry**, where light is split into two beams that travel different paths and recombine to produce interference patterns influenced by gravitational waves.

## About LIGO

LIGO stands for **Laser Interferometer Gravitational-wave Observatory** — the world’s largest gravitational wave observatory.

Major interferometers are located in:
- Hanford, Washington (USA)  
- Livingston, Louisiana (USA)  
- Proposed observatory in Maharashtra, India
